The coffee shop/Triple Espresso

-Descrição geral

Este projeto consiste no desenvolvimento de uma landing page para uma cafeteria fictícia, com foco em layout moderno, tipografia bem definida e organização de conteúdo.

O principal objetivo foi praticar a estruturação semântica com HTML e a estilização com CSS, utilizando conceitos como Flexbox, posicionamento (position), organização com metodologia BEM e manipulação de camadas visuais com z-index. Também foram aplicadas boas práticas de responsividade básica e uso de fontes externas para enriquecer o design.

-Melhorias futuras

Embora o projeto cumpra bem seu propósito visual e estrutural, existem diversas possibilidades de evolução. Uma melhoria importante seria a implementação de responsividade completa, garantindo uma experiência otimizada em dispositivos móveis. Além disso, a inclusão de interatividade com JavaScript — como validação dinâmica do formulário ou animações mais sofisticadas — tornaria a aplicação mais rica.

Outras possíveis evoluções incluem a integração com um backend para tornar o sistema de reservas funcional, a otimização de performance (como carregamento de imagens) e melhorias de acessibilidade, garantindo que o site seja utilizável por um público mais amplo.
